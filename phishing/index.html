Ataque de phishing no Android

Neste artigo, vamos explorar como executar um ataque de engenharia social no sistema operacional móvel Android. No entanto, para aqueles que não estão subordinados, a engenharia social é uma tentativa enganosa de obter informações procedentes, tais como nomes de usuário, senhas e dados de cartão de crédito, através da criação de uma falsa identidade confiável em uma comunicação eletrônica.

Vamos utilizar o aplicativo Termux, que é um emulador de terminal para dispositivos Android e um ambiente Linux, dispensando a necessidade de instalação ou acesso root. O sistema básico mínimo é instalado automaticamente, e é possível obter pacotes adicionais através do gerenciador de pacotes.

Com o Termux, temos à disposição algumas ferramentas que podem auxiliar em testes de invasão no Android. Por exemplo, o Aircrack-ng (um conjunto de utilitários para testes de segurança em redes Wi-Fi), o Hydra (uma ferramenta de força bruta), o Metasploit (uma ferramenta de testes para vulnerabilidades conhecidas) e o Nmap.

Se deseja aprender mais técnicas como essa, convidamos você a conhecer a nossa plataforma de Treinamentos de Cibersegurança.

Para automatizar um ataque de engenharia social, utilizaremos uma ferramenta chamada NEXPHISHER.

  • Primeiro devemos instalar o Termux no Android, disponível na Play Store.
  • Abra o Termux é faça a instalação da ferramenta NEXPHISHER:

    • apt update – Atualiza os repositórios.

    apt install git -y – Instalar o git.

    git clone git://github.com/htr-tech/NEXPHISHER.git – Baixa e instala o NEXPHISHER.

    cd nexphiser – Abra a ferramenta.

    Executamos a ferramenta:

    bash setup

    bash tmux_setup

    bash nexphiser

Você deve ter chegado até essa tela. Então vamos começar o ataque!

  • Então vamos clonar a página de login de uma rede social, por exemplo: digite 1.
  • Escolha a opção “localhostrun”, digite 5.
  • Agora para acessar do navegador basta usar o endereço: http://127.0.0.1:4545/
  • Quando a vítima colocar o e-mail e senha dela nessa tela de login o nexphiser vai capturar.
  • Vale lembrar que a vítima deve estar na mesma rede que a sua.
